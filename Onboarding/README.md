# 인사팀 AI/자동화 역량 강화 프로그램

## 🎯 프로젝트 소개

이 프로젝트는 인사팀 구성원들이 **데이터베이스 조회와 AI 활용 능력**을 습득하여, 일상 업무를 자동화하고 효율화하는 것을 목표로 합니다.

### 왜 이 프로그램이 필요한가?

현재 인사팀의 많은 업무가 엑셀 기반으로 이루어지고 있으며, 다음과 같은 요구사항들이 있습니다:

- **인사 관리**: 주간 인사 발령, 프로젝트 철수 현황 추출 및 인원 현황 최신화
- **급여 관리**: 과거 특정 시점의 본부별/팀별/임원별 인건비 조회, 프로젝트 PL 리스트 관리
- **총무 관리**: 자산 반출/반입 관리, 팀별 자산 조회, 재고 기반 의사결정 및 예측

이러한 업무들을 효율적으로 처리하기 위해서는:
1. **데이터베이스 직접 조회 능력** (SQL)
2. **AI와 협업하는 능력** (효과적인 질문과 프롬프트 작성)
3. **점진적인 자동화 구현** (장기 목표)

## 📚 학습 로드맵

### Phase 1: 기초 역량 구축 (현재 단계)

```
1단계: 환경 설정 (1주차)
  └─ DBeaver 설치 및 DB 연결
  └─ 기본 인터페이스 익히기

2단계: SQL 기초 (2-3주차)
  └─ SELECT, WHERE 기본 조회
  └─ JOIN으로 여러 테이블 연결
  └─ GROUP BY, 집계함수로 통계 산출
  └─ 실습 예제 따라하기

3단계: AI 활용 (4주차)
  └─ AI에게 효과적으로 질문하는 법
  └─ SQL 작성을 AI와 함께 하기
  └─ 프롬프트 작성 팁과 실전 예시

4단계: 실무 적용 (5-6주차)
  └─ 인사 조회 실습
  └─ 급여 조회 실습
  └─ 총무 조회 실습
```

### Phase 2: 자동화 및 고도화 (향후)

- Subagent 개발
- 자동화 워크플로우 구축
- 정기 리포트 자동 생성
- 예측 모델 활용

# 환경 설정 가이드

이 섹션에서는 데이터베이스 조회를 위한 도구를 설치하고 설정하는 방법을 배웁니다.

## 📌 목표

- DBeaver Community Edition 설치
- 데이터베이스 연결 설정
- 기본 인터페이스 익히기

## 📚 학습 자료

1. **[DBeaver 설치 가이드](./DBeaver-설치.md)** ⭐ 필수
   - Windows, macOS 설치 방법
   - 첫 실행 및 기본 설정
   - 데이터베이스 연결하기

## ⏱️ 예상 소요 시간

- 설치: 10분
- 설정 및 연결: 20분
- **총 약 30분**

## ✅ 학습 후 체크리스트

이 섹션을 완료하면 다음을 할 수 있어야 합니다:

- [ ] DBeaver가 컴퓨터에 설치되어 있다
- [ ] 데이터베이스에 성공적으로 연결할 수 있다
- [ ] SQL 편집기를 열 수 있다
- [ ] 간단한 쿼리(예: `SELECT 1`)를 실행할 수 있다

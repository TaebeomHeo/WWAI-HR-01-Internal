# 환경 설정 가이드

이 섹션에서는 데이터베이스 조회를 위한 도구를 설치하고 설정하는 방법을 배웁니다.

## 📌 학습 경로

두 가지 학습 경로 중 선택할 수 있습니다:

### 경로 1: 기초부터 시작 (권장)
DBeaver를 사용해 SQL을 직접 작성하며 학습합니다.
- SQL 기본 개념 이해에 집중
- 단계별로 쿼리 작성 능력 향상
- 초보자에게 권장

### 경로 2: AI 도구 활용
VS Code와 AI를 활용해 자연어로 데이터를 조회합니다.
- 빠른 업무 적용 가능
- AI와 협업하는 방법 학습
- 기본 SQL 지식이 있으면 더욱 효과적

> 💡 **추천**: 경로 1로 시작해 SQL 기초를 익힌 후, 경로 2로 생산성을 높이세요!

## 📚 학습 자료

### 기초 도구 (필수)
1. **[DBeaver 설치 가이드](./DBeaver-설치.md)** ⭐ 필수
   - Windows, macOS 설치 방법
   - 첫 실행 및 기본 설정
   - 데이터베이스 연결하기

### AI 도구 (선택)
2. **[AI 환경 설정 가이드](./AI-환경-설정.md)** 🚀 고급
   - VS Code, Python, Node.js 설치
   - Claude Code Extension 설정
   - 자연어로 DB 조회하기
   - AI 에이전트 활용법

## ⏱️ 예상 소요 시간

**기초 도구**:
- DBeaver 설치 및 설정: 30분

**AI 도구** (선택):
- 전체 환경 설정: 60-90분
- VS Code 설치: 10분
- Python/Node.js 설치: 20분
- Extension 설정: 15분
- 테스트 및 실습: 30분

## ✅ 학습 후 체크리스트

### 기초 도구 체크리스트
- [ ] DBeaver가 컴퓨터에 설치되어 있다
- [ ] 데이터베이스에 성공적으로 연결할 수 있다
- [ ] SQL 편집기를 열 수 있다
- [ ] 간단한 쿼리(예: `SELECT 1`)를 실행할 수 있다

### AI 도구 체크리스트 (선택)
- [ ] VS Code가 설치되어 있다
- [ ] Python과 Node.js가 설치되어 있다
- [ ] Claude Code Extension이 활성화되어 있다
- [ ] 프로젝트를 VS Code에서 열 수 있다
- [ ] AI 에이전트에게 자연어로 질문할 수 있다
- [ ] "지난주 발령 내역 조회해줘" 같은 요청을 실행할 수 있다

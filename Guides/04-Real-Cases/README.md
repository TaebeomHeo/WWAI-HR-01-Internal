# 실제 업무 사례

여기서는 실제 인사/급여/총무 업무에서 필요한 조회 작업을 SQL로 구현해봅니다.

## 📌 목표

- 실무 요구사항을 SQL로 변환하기
- AI와 협업하여 복잡한 쿼리 작성하기
- 데이터 조회 결과를 업무에 활용하기

## 📚 실습 사례

1. **[인사 조회 사례](./인사-조회-사례.md)**
   - 주간 인사 발령 및 프로젝트 현황 추출
   - 부서별/팀별 인원 현황 조회
   - 조직도 데이터 생성

2. **[급여 조회 사례](./급여-조회-사례.md)**
   - 특정 시점의 본부별/팀별/임원별 인건비 조회
   - 월별 프로젝트 PL 리스트 조회
   - 급여 변동 이력 추적

3. **[총무 조회 사례](./총무-조회-사례.md)**
   - 자산 반출/반입 현황 조회
   - 부서별/팀별 자산 현황
   - 재고 기반 예측 및 의사결정

## 💡 학습 방법

각 사례는 다음 구조로 구성되어 있습니다:

1. **업무 요구사항** - 실제로 해결해야 할 문제
2. **테이블 구조** - 필요한 데이터 구조
3. **AI에게 질문하는 법** - 어떻게 질문해야 하는지
4. **해결 과정** - 단계별 쿼리 작성
5. **최종 SQL** - 완성된 쿼리
6. **결과 해석** - 결과를 어떻게 활용하는지

## 🚀 시작하기

실무 요구사항과 가장 가까운 사례부터 시작하세요!

- 급여 업무 👉 **[급여 조회 사례](./급여-조회-사례.md)**
- 총무 업무 👉 **[총무 조회 사례](./총무-조회-사례.md)**

## ✅ 실습 후 체크리스트

이 섹션을 완료하면:

- [ ] 업무 요구사항을 SQL 조회로 변환할 수 있다
- [ ] AI와 대화하며 복잡한 쿼리를 작성할 수 있다
- [ ] 실제 데이터베이스에서 필요한 정보를 추출할 수 있다
- [ ] 조회 결과를 업무에 활용할 수 있다

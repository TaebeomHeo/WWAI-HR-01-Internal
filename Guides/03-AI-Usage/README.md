# AI와 함께 일하기

AI는 SQL 작성, 디버깅, 최적화의 강력한 도구입니다. 이 섹션에서는 AI를 효과적으로 활용하는 방법을 배웁니다.

## 📌 목표

- AI에게 효과적으로 질문하는 방법 배우기
- SQL 작성 시 AI 활용하기
- 프롬프트 작성 기술 익히기
- AI의 답변을 검증하고 개선하기

   - 좋은 질문 vs 나쁜 질문
   - 컨텍스트 제공하기
   - 반복적 개선

2. **[질문 예시](./질문-예시.md)**
   - SQL 작성 요청
   - 오류 해결
   - 최적화 및 개선

   - 명확성

   - 단계별 접근

## 💡 AI 활용의 원칙

### 1. AI는 도우미, 선생님이 아닙니다
- AI의 답변을 그대로 복사하지 말고 이해하세요
- 왜 그렇게 작동하는지 물어보세요
- 스스로 변형해보며 학습하세요

### 2. 항상 검증하세요
- AI도 틀릴 수 있습니다
- 실제 데이터로 테스트하세요
- 결과가 예상과 다르면 다시 질문하세요

### 3. 구체적으로 질문하세요
- 나쁜 예: "SQL 가르쳐줘"
- 좋은 예: "employees 테이블에서 2023년에 입사한 직원의 부서별 평균 급여를 조회하는 SQL을 작성하고, 각 부분을 설명해줘"

## ✅ 학습 후 체크리스트

이 섹션을 완료하면 다음을 할 수 있어야 합니다:

- [ ] AI에게 명확하고 구체적인 질문을 할 수 있다
- [ ] 테이블 구조와 요구사항을 AI에게 설명할 수 있다
- [ ] AI의 답변을 이해하고 실제로 적용할 수 있다
- [ ] 오류가 발생했을 때 AI와 함께 해결할 수 있다
- [ ] AI의 제안을 비판적으로 평가할 수 있다

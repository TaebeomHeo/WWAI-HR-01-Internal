# 사내 업무 포털 자동화 (Portal Automation)

이 디렉토리는 와이즈와이어즈(WiseWires) 사내 업무 포털 내의 작업을 자동화하기 위한 스크립트와 도구를 포함합니다.

## 목적
- 사내 포털에서 반복적으로 수행되는 수동 작업을 자동화합니다.
- HR 및 총무 관련 워크플로우를 효율화합니다.

## 추후 구현 예정 기능 (Planned Features)
1. **결재 대기 건 알림 (Approval Notification)**
    - 승인권자의 결재함을 주기적으로 조회합니다.
    - 미결재 항목이 있을 경우, 해당 내용을 요약하여 결재자에게 통보(메신저/이메일 등)합니다.
    - *목표: 결재 지연 방지 및 업무 속도 향상*

3. **근태 수정 요청 일괄 처리 지원 (Attendance Correction Aggregation)**
    - 근태 수정 게시판을 주기적으로 모니터링합니다.
    - 게시글로 올라온 수정 요청 사항을 엑셀(또는 리스트) 형태로 취합합니다.
    - 취합된 리스트를 통해 일괄 승인 처리가 가능하도록 지원합니다.
    - *목표: 개별 건별 확인/승인하는 단순 반복 업무 제거*

## 구조
- `src/`: 자동화 스크립트 소스 코드 (예: Python/Selenium 등)
- `docs/`: 세부 워크플로우 문서
